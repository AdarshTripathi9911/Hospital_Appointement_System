# Hospital_Appointement_System
The Hospital Appointment Booking System is a Java Spring Boot web app that helps patients book doctor appointments online, check available slots, and manage bookings. Doctors can update schedules, while admins manage doctors, departments, and records securely  A Java Spring Boot web application for managing hospital appointments, patients, doctors,
Features
Patient management
Doctor management
Book appointments
View appointment list
Update appointment status
Create prescriptions
No-login GUI version
H2 database support
Spring Boot MVC architecture

How to Run in Eclipse
Download or clone this project.
Open Eclipse.
Go to File > Import.
Select Maven > Existing Maven Projects.
Choose the project folder.
Click Finish.
Run:
HospitalAppointmentApplication.java
